Bremersee Debian Apt Repository

Add Bremersee's official GPG key:
          
sudo apt update
sudo apt install ca-certificates curl apt-transport-https
sudo install -m 0755 -d /etc/apt/keyrings
sudo curl -fsSL https://debian.bremersee.org/bremersee.asc -o /etc/apt/keyrings/bremersee.asc
sudo chmod a+r /etc/apt/keyrings/bremersee.asc
Add the repository to Apt sources:
          
sudo tee /etc/apt/sources.list.d/bremersee.sources <<EOF
Types: deb
URIs: https://debian.bremersee.org/
Suites: stable
Components: main
Signed-By: /etc/apt/keyrings/bremersee.asc
EOF

sudo apt update
